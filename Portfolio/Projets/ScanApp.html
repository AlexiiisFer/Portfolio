  • ScanApp

    Avril 2024

    Application mobile

    Mon application mobile développée avec Flutter permet aux utilisateurs de scanner les codes-barres des produits alimentaires pour vérifier les allergènes. Elle affiche ensuite clairement les allergènes détectés, fournissant ainsi des informations cruciales pour les personnes souffrant d'allergies.
